Detailed Description

Speaker is the interface for volume and mute control for an AudioChannel.

Volume and mute settings for the Speaker are independent of each other, and the respective directives from the Engine should not affect the other setting in any way.

◆ adjustVolume()

virtual bool aace::alexa::Speaker::adjustVolume ( int8_t  delta)
pure virtual

Notifies the platform implementation to make a relative adjustment to the volume setting of the Speaker. The delta value is relative to the current volume setting and is positive to increase volume or negative to reduce volume. The volume delta value should be scaled to fit the needs of the platform.

Parameters
[in]deltaThe volume adjustment to apply to the Speaker. delta is in the range [-100, 100].
Returns
true if the platform implementation successfully handled the call, else false

◆ localVolumeSet()

void aace::alexa::Speaker::localVolumeSet ( int8_t  volume)

Notifies the Engine of a volume change event originating on the platform, such as a user pressing a "volume up" or "volume down" button. If the Speaker is Type::AVS_SPEAKER_VOLUME, the Engine will respond with a call to setVolume() on each AVS-synced Speaker.

Parameters
[in]volumeThe new volume setting of the Speaker. The volume reported must be scaled to the range [0,100].
