Detailed Description

Alerts should be extended to handle alerts (e.g. timers, alarms, reminders) from AVS. The platform implementation is responsible for rendering visual cues for an active alert. The Alerts MediaPlayer will receive directives from the Engine to handle alerts audio playback.
